# API Module

The API foundation exposes the HTTP routes needed by the framework stage.

Current implementation:

- `src/app.js`
- health checks
- Moling entry route
- current user route
- template list route
- file upload/download routes
- task creation/read routes
- PPT outline, deck, preview, export, slide regeneration, retry, and logs routes
- JSON error responses

- OpenAPI documentation
